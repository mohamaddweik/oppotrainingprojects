# Code Review -> Notion
When this runs:
1. Review the code — the current changes, or the whole
 codebase if asked — for bugs, risks, and cleanup.
2. Write it up: a short summary, then two lists,
 "What's good" and "What to fix" (one line each).
3. Use the Notion tool to create a new page titled
 "Code Review — <project> — <date>" with that write-up.
4. Reply with the link to the Notion page.
