# Figma Design -> Landing Page -> GitHub

When this runs:
1. Look at design-reference.png (and any notes) in this project
   to understand layout, colors, and content.
2. Build a simple, responsive landing page (HTML/CSS, or
   React if the user asks) that matches the design as closely
   as possible.
3. Keep it to a single simple page unless told otherwise.
4. Once built, run: git add ., git commit with a clear message,
   and git push to origin.
5. Reply with a summary of what was built and confirm the push succeeded.
